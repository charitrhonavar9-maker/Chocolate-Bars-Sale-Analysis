# Chocolate Bar Sales Analysis – Power BI Project

### Project Overview

This project analyzes chocolate bar sales data using Microsoft Power BI to uncover business insights, sales trends, and product performance.

### The dashboard helps understand:

* Total Sales Performance
* Profit Analysis
* Top Selling Products
* Regional Sales Trends
* Monthly Revenue Growth
* Customer Purchase Insights


### Tools & Technologies Used

* Power BI
* Power Query
* DAX
* Excel
  

### Key Features

* Interactive Dashboard
* KPI Cards
* Dynamic Filters & Slicers
* Data Cleaning & Transformation
* Sales Trend Analysis
* Profitability Insights


### Business Insights

* Identified high-performing products
* Analyzed monthly sales growth
* Compared regional sales performance
* Tracked profit contribution by category
